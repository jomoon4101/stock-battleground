# 주식 서바이벌 작업완료 보고서 03

작성일: 2026-06-27

## 요청사항 반영 결과

1. 모바일 메인 화면을 단일 열 구조로 다시 설계했습니다. 로고·설명·프로필·게임 모드·버튼의 크기와 여백을 줄이고, 820px 및 520px 이하 전용 레이아웃을 추가했습니다. 모바일에서는 과도한 배경 대형 문자를 숨깁니다.
2. 서비스 표시 이름을 `주식 배틀그라운드`에서 `주식 서바이벌`로 변경했습니다. HTML 제목, 메인 로고, 게임 헤더, 서버 실행 문구, 패키지 이름과 README에 반영했습니다.
3. 최대 플레이어 수를 100명에서 30명으로 변경했습니다. 자동 매칭, 방 정원, AI 충원, 최종 순위와 아이템 순위 범위도 30명 기준입니다.
4. 전체 게임을 40턴에서 25턴으로 변경했습니다. 블라인드 턴은 10·20턴이며 최종 순위는 25턴 종료 시 확정됩니다.
5. `게임 속도`를 `게임 모드`로 바꾸고 공식·빠른·터보를 선택하는 3개 버튼으로 변경했습니다. 25턴 기준 예상 시간은 약 60분·27분·11분입니다.
6. 페이지가 열릴 때 `존버`, `상한가`, `물타기`, `단타` 같은 주식 커뮤니티 단어와 곤충·동물 이름을 조합한 닉네임을 무작위로 생성합니다.
7. 메인 `게임 시작` 버튼을 `멀티게임`으로 변경했습니다.
8. `친구 방 만들기`를 `방 만들기`로 변경했습니다.
9. `혼자서 테스트하기`는 메인 폼 최하단의 작고 흐린 텍스트 링크 형태로 변경했습니다.
10. 게임 화면 하단에 시장·차트·거래·자산·순위·쪽지·규칙 메뉴를 포함한 고정 아이콘 바를 추가했습니다. 각 버튼은 해당 영역으로 이동하거나 모달을 엽니다.
11. 종목별 업종 아이콘을 추가했습니다. 테크, 금융, 소비재, 헬스케어, 산업재, 에너지를 각각 칩·코인·카트·아령·톱니·번개 모양으로 구분합니다.
12. 기업 이름은 방 생성 시 무작위 접두어와 업종형 접미어를 조합하고, 한 방 안에서는 중복되지 않도록 생성합니다. 서버는 방마다 별도 난수 시드를 사용합니다.
13. 미국·한국·일본·중국·유럽에서 각각 3~5개 종목을 생성합니다. 따라서 한 게임의 전체 종목 수는 매번 15~25개 사이에서 달라집니다.
14. 종목 시작 가격 범위를 기존의 10분의 1인 약 1,800~24,000원으로 낮췄습니다.
15. 현재 턴까지 3회 이상 연속 상승한 종목명은 빨간색, 연속 하락한 종목명은 파란색으로 표시하며 연속 횟수도 종목 정보에 표시합니다.
16. 화면 구석에 `A−`, `A＋` 글자 크기 조절 버튼을 추가했습니다. 85~120% 범위에서 조절되며 브라우저 저장소에 선택값을 기억합니다.
17. 게임 상단에 초록색 `OPEN · ROOM` 방 코드 버튼을 추가했습니다. 데스크톱과 모바일 모두 표시되며 누르면 방 코드를 복사하거나 화면 알림으로 확인할 수 있습니다.
18. 메인 페이지 하단에 `일개미(개발자) 메일 : jomoon@gmail.com`을 추가했습니다.

## 규칙 변경에 따른 조정

- 국가별 3~5개라는 조건 때문에 고정 30종목 대신 총 15~25종목을 사용합니다. 다섯 시장이 모두 최소 3종목을 보유하도록 보장합니다.
- 25턴 게임에 맞춰 기존 시간 공식을 그대로 비례 적용해 공식 약 60분, 빠른 게임 약 27분, 터보 게임 약 11분으로 표시했습니다.
- 25턴이 마지막 턴이므로 기존 25턴 월급 사전 알림은 더 이상 의미가 없습니다. 다음 월급이 실제로 남아 있는 15턴에만 `5턴 남음` 알림을 유지합니다.
- 프로젝트 폴더 이름은 기존 실행 경로와 배포 설정을 깨지 않기 위해 `stock-battleground` 그대로 유지하고, 서비스·패키지 표시 이름만 `주식 서바이벌`로 변경했습니다.
