## 완료 항목

1. 카드 클릭 거래창 보강
   - 카드의 일반 `click` 이벤트 외에도 드래그가 발생하지 않은 정상 `pointerup`에서 해당 섹터의 `openStockDetail()`을 직접 실행합니다.
   - 이후 발생하는 중복 click 이벤트는 억제합니다.
   - 실제 좌우 드래그가 발생한 경우에는 거래창이 열리지 않습니다.
   - 카드 왼쪽 위 `거래창 열기` 버튼과 키보드 Enter·Space 진입도 유지했습니다.

2. 생존 정보센터 등락 수치 강조
   - 상승·하락 색상과 굵기를 섹터명이 아닌 실제 등락률 수치에 적용했습니다.
   - 상승 수치는 굵은 붉은색, 하락 수치는 굵은 파란색, 보합 수치는 중립색입니다.

3. 생존 정보센터 3행 구성
   - 1행: `[시장]`
   - 2행: `[공시]`
   - 3행: `[찌라시]`
   - 각 카테고리와 본문이 가로로 섞이지 않도록 독립 행으로 고정했습니다.

4. 최신 정보 우선 표시
   - 각 행의 가장 최신 정보를 왼쪽에 배치했습니다.
   - 두 번째 정보는 약간 어두운 배경과 82% 불투명도, 세 번째 정보는 더 어두운 배경과 68% 불투명도로 표시합니다.
   - 시장 정보는 최근 최대 3턴, 공시는 최신 3건, 찌라시는 최신 3건을 표시합니다.

5. 전체 채팅
   - 하단 고정형 `전체 채팅` 창을 추가했습니다.
   - 멀티플레이 서버에 `send-global-message` 액션을 추가하고 `toId: ALL` 메시지를 모든 참가자에게 브로드캐스트합니다.
   - 닉네임, 메시지, 전송 시각을 표시합니다.
   - 채팅 접기·펼치기와 접힌 상태의 미확인 숫자를 지원합니다.
   - 개인 쪽지 목록에는 전체 채팅이 섞이지 않습니다.
   - 솔로 테스트에서는 AI가 일정 확률로 전체 채팅에 답변합니다.
   - 모바일에서는 하단 메뉴 위 전체 폭으로 표시합니다.

6. 캐시 갱신
   - 앱과 CSS 배포 버전을 `20260701-19`로 갱신했습니다.
