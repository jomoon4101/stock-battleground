# 주식서바이벌 메인 페이지 색상 통일 작업완료 보고서

## 요청 내용

- 메인 페이지에서 푸른색과 흰색 배경이 혼합되지 않도록 하나의 색상 방향으로 통일

## 선택한 방향

- 게임 본편과 자연스럽게 연결되는 푸른색·네이비 테마를 선택했습니다.
- 화이트는 배경 카드가 아니라 제목과 본문 글자에만 사용합니다.
- 민트와 빨강은 핵심 버튼 및 상태 강조에만 제한했습니다.

## 적용 결과

### 전체 배경

- 기존 회백색 배경을 네이비 그라데이션으로 변경했습니다.
- 격자무늬도 밝은 회색 대신 낮은 투명도의 블루 계열로 변경했습니다.

### 메인 설정 영역

- 게임 설정과 진행 중인 서바이벌 카드를 `#172b44` 블루로 통일했습니다.
- 닉네임과 방 코드 입력 영역은 더 짙은 `#10243c`로 구분했습니다.
- 프로필 선택과 게임 모드 버튼은 한 단계 밝은 `#203852`로 구성했습니다.
- 제목은 흰색, 설명은 밝은 청회색으로 통일했습니다.

### 오른쪽 가이드와 명예의 전당

- 흰색 가이드 카드를 설정 카드와 동일한 `#172b44`로 변경했습니다.
- 제목과 순위 이름은 흰색, 설명과 금액은 밝은 청회색으로 조정했습니다.
- 구분선은 블루 테두리로 통일했습니다.

### 실제 브라우저 측정

- 메인 설정 카드: `rgb(23, 43, 68)`
- 가이드 카드: `rgb(23, 43, 68)`
- 입력 영역: `rgb(16, 36, 60)`
- 게임 모드 선택 영역: `rgb(23, 70, 87)`
- 설정·가이드 본문: 흰색
- 프로필 제목: 흰색
