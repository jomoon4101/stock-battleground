# 고대비 주식왕 테마 디자인

## 목표

메인 화면부터 게임·모달까지 낮은 명도 차이와 지나치게 작은 보조 글자를 개선한다. 국가경제교육원 주식왕 페이지의 밝은 정보 카드, 짙은 네이비 핵심 영역, 민트색 행동 버튼을 참고하되 현재 주식서바이벌의 레이아웃과 기능은 유지한다.

## 디자인 원칙

- 메인은 밝은 페이지 위에 짙은 네이비 소개 카드와 흰색 설정·진행방 카드를 배치한다.
- 게임 화면은 집중도를 위해 네이비를 유지하되, 패널 표면과 배경의 명도 차이 및 테두리 대비를 높인다.
- 핵심 행동은 민트색, 상승은 빨강, 하락은 파랑, 경고는 노랑으로 역할을 고정한다.
- 본문은 최소 14px, 라벨은 최소 12px 수준으로 올리고 보조 글자도 회색 대신 밝은 청회색을 사용한다.
- 텍스트 선택 색상도 민트색으로 바꿔 파란 선택 배경에서 글자가 뭉개지는 현상을 막는다.

## 구조

기존 `styles.css`, `mobile-first.css`, `design-system.css`는 보존한다. 마지막에 로드되는 `high-contrast-theme.css`가 색상 토큰과 핵심 컴포넌트만 덮어쓴다. 기능 코드와 DOM 구조를 바꾸지 않아 게임 로직 회귀 가능성을 낮춘다.

## 적용 범위

- 메인: 소개 영역, 게임 설정, 진행 중 목록, 규칙 카드, 입력·버튼
- 게임: 상단 HUD, 프로필, 자산·보유 종목·대체자산, 섹터 시장, 정보센터, 주문 영역, 하단 메뉴
- 오버레이: 모달, 바텀시트, 토스트, 채팅·쪽지
- 반응형: 모바일 우선, 768px과 1180px에서 여백과 카드 구조 확장

## 검증

- 계약 테스트로 로드 순서, 빌드 포함, 서비스 워커 캐시, 핵심 선택자를 확인한다.
- 전체 Node 테스트와 프로덕션 빌드를 실행한다.
- 메인과 게임 화면을 브라우저에서 확인해 글자 겹침, 대비, 버튼 상태를 점검한다.
