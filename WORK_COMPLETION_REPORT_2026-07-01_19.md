# 주식 서바이벌 작업완료 보고서

작성일: 2026-07-01

## 완료 항목

### 생존정보센터

- 첫 줄: `[시장]`
- 둘째 줄: `[공시]`
- 셋째 줄: `[찌라시]`
- 세 영역을 같은 왼쪽 기준으로 정렬했습니다.
- 각 줄 내부의 최신 정보 우선 배치와 이전 정보 어둡게 표시 기능은 유지했습니다.

### AI 전체 채팅

- AI가 게임 시작, 턴 전환, 사람의 전체 채팅 이후에 대화할 수 있도록 구현했습니다.
- 한 번에 2~4명의 AI가 앞 발언을 참고해 순차적으로 답합니다.
- AI의 보유 종목을 우선 주제로 선택하고, 없으면 현재 시장 종목 중 하나를 선택합니다.
- 찬성, 반박, 질문, 관망, 거래량 의심 등 여러 대화 유형을 추가했습니다.
- 최근 AI 문장과 같은 문장은 다시 선택하지 않도록 중복 방어를 적용했습니다.
- 다음 가격 방향을 기준으로 약 62%는 맞는 의견, 약 38%는 반대 방향의 거짓·오판 정보를 말합니다.
- 거짓 정보 여부는 플레이어 화면에 표시하지 않아 실제 게임 정보전처럼 판단해야 합니다.
- 한국어 종목명 받침에 따라 `은/는`, `이/가` 조사를 자동 보정합니다.
- 영어 모드용 대화도 함께 구현했습니다.
- AI에게 보내는 1대1 쪽지도 기존 고정 답변 대신 새 맥락형 답변을 사용합니다.

## 구현 파일

- `ai-chat.js`: AI 대화와 정보 진위 생성기
- `app.js`: 혼자 테스트 모드 AI 대화 연결
- `server.mjs`: 멀티게임 AI 간 대화 연결
- `mobile-first.css`: 생존정보센터 3행 레이아웃
- `scripts/build.mjs`: 배포 산출물에 AI 대화 모듈 포함
- `tests/ai-chat.test.js`: 거짓 정보와 AI 간 연속 대화 검증

## 검증 결과

- JavaScript 구문 검사 성공
- 자동 테스트 38개 전체 통과
