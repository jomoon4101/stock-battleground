# 주식서바이벌 섹터 카드 상태 이미지 수정 보고서

## 요청 내용

- 섹터 카드 안에서 현재 감정 상태 외의 다른 상태 카드 이미지가 옆에 같이 보이는 문제 수정

## 원인

- 섹터 CEO 원본 이미지는 기본·상승·하락 이미지가 가로로 이어진 3칸 스프라이트입니다.
- 카드형 레이아웃에서 이미지 크기를 `cover`로 덮어쓰면서 스프라이트 전체가 카드에 표시됐습니다.
- 기존 상태 위치 계산(`0%`, `50%`, `100%`)은 정상적으로 동작하고 있었지만, 프레임 크기가 잘못되어 옆 상태가 노출됐습니다.

## 수정 내용

- 일반 섹터 카드는 `background-size: 300% 100%`로 복원했습니다.
- 이미지 반복을 `no-repeat`로 고정했습니다.
- IT 섹터는 세로로 긴 원본 비율을 유지하기 위해 `background-size: 300% auto`를 적용했습니다.
- 섹터 상세 거래창의 작은 CEO 이미지에도 같은 프레임 분리 규칙을 적용했습니다.
- 기존 상태 위치 선택 로직은 변경하지 않았습니다.

## 검증 결과

- 기본 상태: 첫 번째 프레임만 표시
- 상승 상태: 가운데 프레임만 표시
- 하락 상태: 마지막 프레임만 표시
- 실제 브라우저에서 금융·부동산·산업재·에너지 등 카드의 계산된 스타일이 `300% 100% / no-repeat`인 것을 확인
- IT 카드는 `300% auto` 적용 확인
- 전용 회귀 테스트 추가
- 전체 테스트와 프로덕션 빌드 실행
