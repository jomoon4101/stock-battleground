# 주식서바이벌 모바일 우선·섹터 카드 개편 작업완료 보고서

- 작업일: 2026-07-01
- 최종 상태: 요청 9개 항목 반영 완료
- 검증 상태: 자동 테스트 30개 통과, 배포 빌드 성공, 모바일·웹 실제 브라우저 검증 완료

## 1. 모바일 우선 레이아웃

- `mobile-first.css`를 새로 만들고 기존 스타일 다음에 적용했다.
- 기본 CSS는 320px 이상 모바일 단일 열 구조로 작성했다.
- 900px 이상에서만 `@media (min-width: 900px)`로 웹 그리드 레이아웃을 활성화한다.
- 모바일 메인 폼, 게임 모드, 섹터 카드, 상태바, 거래·자산·순위 영역을 단일 열 흐름으로 정리했다.
- 웹에서는 메인 안내 패널, 3열 게임 모드, 4열 섹터 카드, 중앙 작업 공간과 우측 순위 패널로 확장된다.

## 2. 메인 설정 단순화

- 생존 인원 선택 UI를 삭제했다.
- 투자 구역 난이도 선택 UI를 삭제했다.
- 게임 모드만 선택하면 서버와 클라이언트가 인원·라운드·섹터 수를 자동 결정한다.
- 언어 선택을 메인 상단의 작은 `ko` / `En` 버튼으로 이동했다.
- 언어 버튼 표기는 번역 과정에서도 변경되지 않도록 고정했다.

## 3. 게임 모드 자동 규칙

- 빠른 게임: 3명, 10라운드, 5섹터
  - 민감 섹터 3개
  - 안정 섹터 2개
- 기본 게임: 5명, 20라운드, 8섹터
  - 민감 섹터 5개
  - 안정 섹터 3개
- 장기 게임: 7명, 30라운드, 11개 전체 섹터
- 생산 서버는 요청 본문의 임의 인원·난이도 값을 무시하고 선택 모드 규칙을 사용한다.
- 멀티플레이 자동 매칭도 같은 모드끼리만 결합한다.

## 4. 11개 섹터와 고정 시작가

| 섹터 | 시작가 | 수익성 | 안정성 | 변동성 |
|---|---:|---|---|---|
| 정보기술 | 120 | 높음 | 낮음 | 높음 |
| 금융 | 100 | 중간 | 중간 | 중간 |
| 헬스케어 | 95 | 중간 | 높음 | 중간 |
| 경기소비재 | 110 | 높음 | 낮음 | 높음 |
| 필수소비재 | 80 | 낮음 | 높음 | 낮음 |
| 산업재 | 90 | 중간 | 중간 | 중간 |
| 커뮤니케이션 서비스 | 105 | 높음 | 낮음 | 높음 |
| 원자재 | 85 | 중간 | 낮음 | 높음 |
| 에너지 | 115 | 높음 | 낮음 | 높음 |
| 유틸리티 | 75 | 낮음 | 높음 | 낮음 |
| 부동산 | 90 | 중간 | 중간 | 중간 |

- 한 게임에서 같은 섹터가 중복 생성되지 않는다.
- 각 섹터에는 정확히 한 개의 익명 회사만 생성된다.
- 국가 목록, 국기, 국가 필터와 국가 기반 종목 표기를 제거했다.
- 찌라시와 AI 대화도 국가 대신 섹터를 사용한다.
- `주식 종목` 중심 문구를 `주식 섹터` 중심으로 변경했다.

## 5. CEO 캐릭터 카드

- 종목 목록을 섹터별 대형 카드로 변경했다.
- 카드 중앙에 해당 섹터 CEO 캐릭터를 표시한다.
- 카드 배경에 섹터별 색상과 관련 산업 모티프를 적용했다.
- 카드에 섹터명, 회사명, 설명, 현재가, 등락률, 수익성, 안정성, 변동성을 표시한다.
- 보합: 기본 자세
- 상승: 양팔을 들고 환호하는 표정과 자세
- 하락: 어깨를 숙인 우울한 표정과 자세
- 2턴 실제 검증에서 상승 카드 3개, 하락 카드 2개로 캐릭터 상태가 바뀌는 것을 확인했다.

## 6. 생성 이미지 자산

- 생성 방식: Codex 기본 `image_gen` 도구
- 생성 자산:
  - `assets/sector-ceo-atlas-v1.png`
  - `assets/sector-ceo-real-estate-v1.png`
- 첫 생성물은 정보기술부터 유틸리티까지 10개 섹터의 기본·환호·우울 상태를 한 아틀라스로 구성했다.
- 생성 결과에 부동산 행이 누락되어 부동산 CEO 3상태 이미지를 별도로 생성해 보완했다.
- 이미지 프롬프트 핵심:
  - 모바일 전략 보드게임용 2D 캐릭터 일러스트
  - 같은 CEO가 기본·환호·우울 3상태로 등장
  - 섹터별 의상과 배경 모티프
  - 작은 카드에서도 표정과 자세가 명확함
  - 텍스트·로고·워터마크 없음

## 7. 한국어·영어
