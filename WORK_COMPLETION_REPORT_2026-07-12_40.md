# 작업완료 보고서 — 대체자산 카드 글자 겹침 수정

## 원인

- 대체자산 목록이 3열 카드로 변경된 뒤에도 카드 내부 버튼이 이전 레이아웃의 세 번째 열을 점유했습니다.
- 좁아진 카드에서 아이콘, 자산명, 수량과 증감 버튼의 가로 공간 합계가 카드 폭보다 커져 서로 겹쳤습니다.

## 반영 내용

- 각 대체자산 카드를 2열·3행 내부 구조로 변경했습니다.
- 첫째 행은 `아이콘 + 자산명/수량`, 둘째 행은 가격, 셋째 행은 `+ / -` 버튼으로 분리했습니다.
- 증감 버튼 영역을 카드 전체 폭으로 배치하고 버튼 폭을 균등하게 나눴습니다.
- 긴 자산명은 버튼을 침범하지 않고 말줄임표로 표시되도록 처리했습니다.
- 금, 구리, 코인 카드 모두 동일한 정렬 규칙을 사용하도록 통일했습니다.
