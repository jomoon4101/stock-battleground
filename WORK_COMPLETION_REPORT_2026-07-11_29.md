- 제한시간 종료 시 배정된 스킬 중 두 개를 자동 장착한 뒤 방어 행동으로 안전하게 진행합니다.

## 2. 실제 점검 중 발견해 수정한 문제

- 모바일 하단 메뉴와 턴 행동 바가 이전 중앙 정렬 속성을 상속해 왼쪽으로 밀리던 문제
- 턴 상세 패널이 고정 배치되어 시장·순위 화면을 가리던 문제
- 스킬 미선택 상태에서 제한시간 종료 알림이 반복되던 문제
- 기존 선택자 우선순위 때문에 섹터 카드가 흰색으로 표시되던 문제
- 데스크톱에서도 앱 루트가 480px로 제한되던 문제
- 데스크톱 턴 행동 바가 콘텐츠 중심보다 52px 오른쪽으로 치우치던 문제

각 문제에는 재발 방지 테스트를 추가했습니다.
