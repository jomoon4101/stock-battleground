# 프로필 선택·제한시간·섹터 한 화면 개편 작업완료 보고서

작성일: 2026-07-01

## 1. 메인 화면 프로필 선택 개선

- 메인 화면에 10개 프로필을 모두 노출하던 구조를 제거했습니다.
- 현재 선택한 프로필 아이콘 하나와 `프로필 선택` 버튼만 표시합니다.
- 버튼을 누르면 전용 프로필 선택 창이 열립니다.
- 선택 창에서 기본 주식 밈 프로필 10개를 선택할 수 있습니다.
- 기존 사진 업로드 기능도 선택 창 안으로 이동했습니다.
- `선택 완료`, 닫기 버튼, 배경 클릭, ESC 키로 창을 닫을 수 있습니다.
- 프로필을 바꾸면 메인 화면의 현재 프로필 아이콘이 즉시 갱신됩니다.
- 한국어와 영어 인터페이스 문구를 함께 추가했습니다.

## 2. 이전 버전 제한시간 복원

- 빠른 게임: 턴당 45초
- 기본 게임: 턴당 90초
- 장기 게임: 턴당 120초
- 매 10번째 턴: 30초 추가
- 10·20·30번째 턴 종료 10초 전: 상단 카운트다운 표시
- 혼자서 테스트하기: 시간이 끝나면 자동으로 다음 턴 진행
- 멀티게임: 참가자 전원이 먼저 `턴 종료`를 누르거나 서버 제한시간이 끝나면 다음 턴 진행
- 혼자서 테스트하기에는 일시정지 버튼과 스페이스바 일시정지를 복원했습니다.
- 서버가 각 방에 `deadline`과 `durationSeconds`를 제공하도록 복원했습니다.
- 서버의 제한시간 만료 검사와 자동 턴 정산을 복원했습니다.

## 3. 주식 섹터 한 화면 구성

- 5·8·11개 섹터 모두 같은 가로 카드 레일 안에 표시합니다.
- 종목 정보, CEO 3상태, 보유수량, 평균매수가, 평가손익, 섹터 특성, 미니 차트는 유지했습니다.
- 카드 높이를 기존 약 650px에서 모바일 382px, 데스크톱 405px로 축소했습니다.
- 모바일에서는 손가락으로 좌우 스와이프할 수 있습니다.
- PC에서는 좌우 이동 버튼, 트랙패드, 가로 스크롤로 탐색할 수 있습니다.
- 카드 선택이나 화면 재렌더링 후에도 가로 스크롤 위치를 유지합니다.
- 카드는 한 장씩 자연스럽게 멈추는 스크롤 스냅을 사용합니다.
- 시장 패널이 화면 전체 높이를 차지하지 않도록 고정 높이 구조를 제거했습니다.

## 4. 주요 수정 파일

- `index.html`: 단일 프로필 버튼, 프로필 모달, 제한시간 UI, 카운트다운, 일시정지, 섹터 이동 버튼
- `app.js`: 프로필 모달 상태, 타이머, 자동 턴 종료, 일시정지, 섹터 레일 이동 및 위치 보존
- `engine.js`: 모드별 턴 시간과 10턴 추가시간 복원
- `server.mjs`: 방 마감시간, 자동 턴 정산 복원
- `mobile-first.css`: 프로필 모달·타이머·컴팩트 가로 섹터 레일 디자인
- `i18n.js`: 새 프로필·제한시간 문구 영어 번역
- `README.md`: 제한시간 규칙 문서 복원
- `tests/engine.test.js`, `tests/multiplayer.test.js`, `tests/ui-contract.test.js`: 제한시간·자동진행·프로필·가로 레일 검사

## 6. 적용 시 주의사항

- 현재 4173 포트에는 기존 버전 서버와 진행 중인 방 1개가 남아 있어 해당 방을 강제로 종료하지 않았습니다.
- 새 서버 제한시간 로직을 적용하려면 기존 `게임실행.cmd` 창을 닫은 뒤 다시 실행해야 합니다.
- 서버를 재시작하면 메모리에서 진행 중이던 기존 방은 사라집니다.
- 인앱 브라우저의 localhost 자동 조작 차단 설정 때문에 실제 탭 시각 검증은 실행하지 않았고, 정적 UI 계약·서버 통합 테스트·배포 빌드로 검증했습니다.
