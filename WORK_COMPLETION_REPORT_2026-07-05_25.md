## 반영 내용

1. 메뉴/패널 흰 배경 수정
   - `.panel`, `.game-card`, `.panel-header` 계열의 최종 스타일을 다크 UI 기준으로 통일했습니다.
   - 기존에 뒤쪽 CSS에서 다시 흰색으로 덮어쓰던 레거시 스타일을 정리했습니다.

2. 알림 숫자 배지 확대
   - `#mail-unread`, `#notice-unread`, `#global-chat-unread` 배지를 최소 18px 크기와 굵은 숫자로 조정했습니다.
   - 버튼 우측 상단에서 잘 보이도록 위치와 테두리를 정리했습니다.

3. 섹터 카드 버튼/이름 정리
   - 카드 내부의 중복 `거래창 열기` 버튼을 제거했습니다.
   - 카드 자체 클릭으로 거래창이 열리는 기존 동작은 유지했습니다.
   - 섹터 이름을 카드 좌측 상단 오버레이 영역으로 이동해 이미지와 겹치는 문제를 줄였습니다.

4. 생존정보센터 재구성
   - 정보센터를 3행 구조로 고정했습니다.
     - 1행: `[시장]`
     - 2행: `[공시]`
     - 3행: `[찌라시]`
   - 각 행은 최신 정보 1개만 표시합니다.
   - 오래된 정보 목록은 화면에 길게 쌓이지 않도록 제거했습니다.
   - 상승/하락 수치는 기존 색상 체계를 유지해 상승은 붉은색, 하락은 푸른색으로 표시됩니다.
