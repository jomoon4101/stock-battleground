# 주식서바이벌 순수 턴제 및 디자인 개선 작업완료 보고서

작성일: 2026-07-01

## 1. 순수 턴제 전환

- 게임 라운드의 제한시간을 완전히 제거했습니다.
- 시간이 지나도 턴이 자동으로 종료되지 않습니다.
- 혼자서 테스트하기에서는 사용자가 `턴 종료`를 눌렀을 때만 다음 턴으로 진행합니다.
- 온라인 게임에서는 탈락하지 않은 일반 플레이어 전원이 `턴 종료`를 눌렀을 때만 서버가 다음 턴을 정산합니다.
- 10·20·30턴의 추가시간과 종료 10초 전 카운트다운을 제거했습니다.
- 일시정지 버튼과 스페이스바 일시정지 기능을 제거했습니다. 시간 흐름 자체가 없기 때문에 더 이상 필요하지 않습니다.
- 상단 시간 표시는 `∞`, `순수 턴제 · 시간 무제한` 안내로 변경했습니다.
- 온라인 상단 진행 바는 남은 시간이 아니라 `준비 완료 인원 / 생존 일반 플레이어` 진행률을 표시합니다.
- 생존 상태바의 `남은 시간`을 `진행 방식 · 무제한`으로 교체했습니다.

## 2. 서버 진행 규칙

- 방 상태에서 게임 턴 마감시간과 턴 길이 값을 제거했습니다.
- 서버의 마감시간 검사와 자동 `advanceTurn` 실행을 제거했습니다.
- `end-turn` 요청으로 모든 생존 일반 플레이어가 준비된 경우에만 `advanceRoom`을 실행합니다.
- 기존 코드와 외부 참조의 호환성을 위해 `turnDurationSeconds()` 함수는 유지하되 항상 `null`을 반환합니다.
- 매칭 시작 후 5초 동안 플레이어를 모으고 AI로 빈자리를 채우는 로비 기능은 기존 요구사항이므로 유지했습니다. 이 시간은 게임 턴 제한시간이 아닙니다.

## 3. 디자인 개선

- 전체 색상 체계를 딥 네이비, 스틸 블루, 레드 포인트 중심으로 통일했습니다.
- 메인 화면에 미세한 금융 그리드와 부드러운 블루·레드 조명 배경을 추가했습니다.
- 시작 폼, 진행 중 게임 목록, 게임 규칙 영역을 반투명 글래스 카드로 개선했습니다.
- 게임 모드 버튼에 선택 표시선, 입체 그림자, 호버 이동 효과를 적용했습니다.
- 상단바를 반투명 네이비 그라데이션과 유리 효과로 재디자인했습니다.
- 순수 턴제 상태 영역에 무제한 표시와 온라인 준비 진행 바를 추가했습니다.
- 생존 정보 영역을 개별 상태 카드 형태로 분리했습니다.
- `턴 종료` 버튼의 대비, 깊이감, 호버 반응을 강화했습니다.
- 패널, 종목 카드, 차트, 포트폴리오, 하단 메뉴의 테두리·그림자·라운드 값을 통일했습니다.
- 종목 카드 선택 및 호버 상태를 더 명확하게 표시합니다.
- 모바일 우선 구조와 900px 이상 데스크톱 전환 구조는 그대로 유지했습니다.

## 4. 주요 수정 파일

- `engine.js`: 모드별 턴 시간 제거 및 시간 함수 호환 처리
- `server.mjs`: 턴 마감시간과 자동 진행 제거, 준비 인원 상태 추가
- `app.js`: 타이머·카운트다운·일시정지 제거, 턴제 상태 렌더링 추가
- `index.html`: 시간 UI를 턴제 상태 UI로 교체
- `mobile-first.css`: 메인·게임 화면 디자인 시스템 개선
- `i18n.js`: 한국어·영어 턴제 안내 추가
- `README.md`: 시간제한 없는 턴제 규칙으로 문서 수정
- `tests/engine.test.js`: 시간제한 제거 계약 검사
- `tests/multiplayer.test.js`: 전원 턴 종료 방식의 서버 진행 검사
- `tests/ui-contract.test.js`: 타이머·서버 마감시간 부재와 디자인 계약 검사
