# 주식서바이벌 고대비 UI 작업완료 보고서

## 요청 사항

1. 메인부터 게임 전반까지 글자와 배경색이 비슷해 읽기 어려운 문제 해결
2. 국가경제교육원 주식왕 페이지를 참고한 웹사이트 분위기 개선

## 적용 결과

### 1. 고대비 색상 체계

- 게임용 짙은 네이비 배경, 밝은 청회색 보조 글자, 선명한 흰색 본문으로 색상 역할을 정리했습니다.
- 패널 테두리를 이전보다 밝게 만들고 카드 표면을 단계별 네이비로 구분했습니다.
- 핵심 행동 버튼은 민트, 상승은 빨강, 하락은 파랑, 중요 정보는 노랑으로 통일했습니다.
- 파란 텍스트 선택 배경을 민트색으로 변경해 선택 중인 글자도 읽을 수 있게 했습니다.
- 작은 HUD·순위·로그·쪽지 라벨의 최소 글자 크기와 색상 대비를 높였습니다.

### 2. 메인 페이지 분위기

- 밝은 회백색 페이지 배경과 짙은 네이비 소개 카드를 조합했습니다.
- 게임 설정과 진행 중인 서바이벌은 흰색 카드로 분리해 입력 영역을 바로 찾을 수 있게 했습니다.
- 오른쪽 게임 안내·명예의 전당 카드는 흰색 정보 카드로 유지하고 본문을 진한 남색으로 변경했습니다.
- 활성 언어와 주요 시작 버튼은 민트색으로 바꿔 행동 우선순위를 명확하게 만들었습니다.

### 3. 게임 화면과 오버레이

- 상단 HUD, 프로필, 자산, 섹터, 정보센터, 거래·순위·로그 패널의 경계를 강화했습니다.
- 하단 메뉴의 활성 탭을 민트색 면으로 표시해 현재 위치를 한눈에 알 수 있게 했습니다.
- 채팅, 쪽지, 모달, 바텀시트, 토스트에도 같은 고대비 표면과 테두리를 적용했습니다.
- 모바일 우선 스타일을 유지하고 768px·1180px 구간에서 여백과 패널 크기를 확장합니다.

## 구현 구조

- `high-contrast-theme.css`를 기존 `design-system.css` 다음에 로드하는 독립 테마 레이어로 추가했습니다.
- 기존 게임 DOM, 상태, 이벤트, 엔진 로직은 변경하지 않았습니다.
- 빌드 산출물과 서비스 워커 오프라인 캐시에 새 테마를 포함했습니다.
- 서비스 워커 캐시 버전을 `v32`로 올려 배포 후 이전 CSS가 남지 않게 했습니다.

## 검증 결과

- 고대비 테마 계약 테스트: 3/3 통과
- 전체 자동 테스트: 137/137 통과
- 프로덕션 빌드: 성공 (`dist` 생성)
- 브라우저 메인 화면: 네이비 소개·흰 설정·흰 안내 카드의 구분과 본문 대비 확인
- 브라우저 게임 화면: HUD·패널·버튼·보조 글자 대비 및 민트 활성 메뉴 확인

## 참고 디자인

- 국가경제교육원 주식왕 페이지의 네이비 핵심 영역, 밝은 정보 섹션, 민트색 행동 버튼, 여백 중심 카드 구성을 분위기 참고 자료로 사용했습니다.
- 사이트를 복제하지 않고 현재 주식서바이벌의 구조와 게임성을 유지한 별도 테마로 재구성했습니다.
