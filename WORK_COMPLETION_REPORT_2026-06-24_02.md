# 주식 배틀그라운드 작업완료 보고서 02

작성일: 2026-06-24

## 요청사항 반영 결과

1. 생존 순위 오른쪽 영역을 324px에서 370px로 넓히고 순위 행을 5열 고정 구조로 수정했습니다. 자산 금액은 최소 폭을 확보하고 글자를 11px 굵은 고정폭 글꼴로 키워 겹침과 잘림을 없앴습니다.
2. 쪽지함에는 실제로 받은 쪽지와 보낸 쪽지가 존재하는 상대만 표시됩니다. `쪽지 보내기` 버튼을 누르면 별도의 플레이어 검색·선택 영역이 열립니다.
3. 일반 플레이어에게 1·11·21·31턴에 `찌라시` 개인 쪽지가 발송됩니다. 현재 턴부터 7턴 이내의 실제 가격 데이터를 사용해 방향을 정하지만, 본문에는 `[나라]`, `[종목]`과 구어체 암시만 넣고 발생 턴과 변동 폭은 숨겼습니다.
4. AI 답장 문구를 일반 대화 12종과 보유 종목 대화 8종으로 확장했습니다. 종목을 언급할 때는 해당 AI의 현재 보유 종목 중 하나만 사용합니다.
5. 10·20·30·40턴 또는 순위 차단 아이템 적용 턴에는 1위 교체·탈환 알림이 생성되거나 뒤늦게 표시되지 않도록 서버와 클라이언트 양쪽에서 차단했습니다.
6. 진행 중인 친구 방에도 방 코드로 참가할 수 있습니다. 새 참가자는 현재 생존 중인 AI 좌석 하나를 이어받고 기존 방의 턴과 자산 상태로 즉시 참여합니다.
7. 최근 5개 봉 평균 대비 거래량이 1.65배 이상이면 폭등, 0.62배 이하면 폭락으로 감지합니다. 차트 거래량 막대에 빨강·파랑 방향 마커를 그리고 현재 봉에는 상태 배지를 표시합니다.
8. 영어 모드는 고정 환율 `1 USD = 1,500 KRW`을 적용합니다. 화면의 가격·자산·월급·주문·대출·채권·로그를 달러로 바꾸고, 지정가·대출·채권 입력값은 서버 전송 전에 원화 기준값으로 역환산합니다.
9. 15·25·35턴 시작 시 모든 일반 플레이어에게 `월급날까지 5턴 남았습니다.` 알림을 표시합니다.
10. 플레이어 탈락 시 모든 참가자에게 이름이 포함된 `A플레이어가 깡통을 찼습니다.` 형식의 알림을 표시합니다. 본인의 탈락 화면 제목에도 닉네임을 표시합니다.

## 구현 판단과 대체 방식

- 중도 참가자에게 새 자산을 임의 지급하면 기존 참가자와 순위 계산이 불공정해지므로, 생존 AI 좌석을 인계하는 방식으로 구현했습니다.
- `찌라시`는 외부 뉴스나 실제 투자 정보를 사용하지 않습니다. 게임 시작 시 이미 생성된 가상 시세의 7턴 구간만 분석하므로 정보가 게임 안에서만 유효합니다.
- AI 답장은 외부 LLM 호출 대신 충분히 다양한 구어체 템플릿과 AI 보유 종목 데이터를 조합합니다. API 키·비용·개인정보 전송 없이 응답하며 종목 관련 발언의 사실관계를 게임 상태와 일치시킬 수 있습니다.
- 환율은 요청대로 1,500원 고정값입니다. 실제 환율 API를 사용하지 않으므로 게임 중 환율 변동은 없습니다.
