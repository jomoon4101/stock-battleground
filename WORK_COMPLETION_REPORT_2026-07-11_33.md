# 주식서바이벌 상단 HUD 정리 작업완료 보고서

## 요청 내용

1. 상단 아이콘과 글자가 겹치는 문제 수정
2. 현재 라운드·생존 순위·생존 자산·보유 현금·생존 위협도·남은 시간 영역을 글자 크기에 맞게 조정

## 원인

- 로고 아이콘이 상단 그리드의 남은 폭에 따라 축소될 수 있어 옆 글자 영역을 침범했습니다.
- 이전 반응형 규칙의 상태바 최소 높이가 데스크톱에도 남아 있어, 실제 두 줄 글자보다 카드가 과도하게 높았습니다.
- 데스크톱 상단 그리드가 로고 영역의 최소 폭을 보장하지 않아 긴 닉네임과 진행 정보가 서로 밀릴 수 있었습니다.

## 수정 내용

- 로고 아이콘을 `40px` 고정 폭으로 설정해 축소되지 않게 했습니다.
- 로고 글자 영역에 말줄임 처리를 적용해 진행 막대 쪽으로 침범하지 않게 했습니다.
- 데스크톱 로고 열에 최소 `210px`을 보장했습니다.
- 상태바의 기존 최소 높이를 초기화하고 각 카드 높이를 데스크톱 `62px`로 조정했습니다.
- 6개 상태 카드는 각각 최소 `132px`을 확보하면서 남은 공간을 균등하게 사용합니다.
- 모바일 기본 상태 카드는 `58px` 높이를 사용해 두 줄 정보가 잘리지 않도록 유지했습니다.

## 실제 화면 검증

- 로고 아이콘: `40 × 40px`
- 로고와 글자 겹침: 없음 (`overlap: false`)
- 상태 카드: 6개 모두 약 `170.66 × 62px`
- 전체 상태바: `1053 × 70px`
- 현재 라운드부터 남은 시간까지 글자 잘림 없이 표시됨
