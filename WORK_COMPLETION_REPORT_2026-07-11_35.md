# 주식서바이벌 메뉴·아이콘·프로필 개선 작업완료 보고서

## 요청 내용

1. 데스크톱 좌측 고정 메뉴를 상단으로 이동하고 메뉴별 아이콘 교체
2. `0개 종목`, `보유 종목 없음`, 금·구리·코인 0개 상태의 가독성 개선
3. 금·구리·코인·세무조사·공매도 아이콘을 서로 다르게 제작
4. 프로필 이미지가 보이지 않는 문제 수정 및 MY TRADER 상단 고정

## 수정 결과

### 좌측 고정 메뉴

- 메뉴 정렬을 화면 중앙에서 상단 정렬로 변경했습니다.
- 상단 HUD와 겹치지 않도록 데스크톱 시작 위치를 `84px`로 설정했습니다.
- 홈은 집, 시장은 막대 차트, 거래는 양방향 화살표, 순위는 트로피, 로그는 문서 형태의 SVG 아이콘을 사용합니다.
- 활성 메뉴에서는 아이콘과 배경색이 함께 변경됩니다.

### 빈 상태와 0개 상태

- `0개 종목`은 흰색 14px로 표시합니다.
- `보유 종목 없음`은 흰색 16px로 표시합니다.
- 금·구리·코인 보유량은 흰색 14px로 표시합니다.
- 보유 스킬카드가 없을 때 안내문은 밝은 청회색 13px로 표시합니다.
- 금·구리·코인의 초기 문구를 각각 `금 0개`, `구리 0개`, `코인 0개`로 명확하게 변경했습니다.

### 자산·스킬 아이콘

- 금: 금괴 모양과 노란색
- 구리: 동전·코일 모양과 구리색
- 코인: 디지털 코인 모양과 보라색
- 세무조사: 문서와 돋보기 모양, 민트색
- 공매도: 하락 차트 모양, 붉은색
- 내부정보와 기타 스킬도 별도 눈·별 아이콘을 사용하도록 확장했습니다.

### 프로필과 MY TRADER

- 프로필 버튼의 배경 속성이 밈 스프라이트 이미지를 지우던 CSS 충돌을 수정했습니다.
- 업로드 프로필과 기본 밈 프로필을 모두 기존 방식으로 표시합니다.
- 데스크톱 MY TRADER 카드를 HUD 아래 `142px` 위치에서 고정해 스크롤 중에도 정보를 확인할 수 있습니다.

## 실제 브라우저 확인

- 고정 메뉴 SVG 아이콘: 5개
- 대체자산 SVG 아이콘: 3개
- 프로필 이미지 URL 정상 적용 확인
- 금·구리·코인·포트폴리오 수량: 흰색 14px 확인
- 좌측 메뉴 시작 위치: 화면 상단 84px
- MY TRADER 시작·고정 위치: HUD 아래 영역
