# 주식 서바이벌 작업완료 보고서 14

작성일: 2026-07-01

## 완료 항목

1. 거래창 안의 섹터 선택 UI
   - 거래 모달 상단에 현재 방의 모든 섹터를 가격·등락률과 함께 보여주는 가로 선택 목록을 추가했습니다.
   - 다른 섹터를 누르면 모달을 닫지 않고 해당 섹터의 차트와 매수·매도 화면으로 즉시 전환됩니다.

2. 카드의 거래창 버튼 위치와 구조
   - `거래창 열기`를 카드 왼쪽 위로 이동했습니다.
   - 카드 전체 클릭과 별개인 실제 `button` 요소로 만들어 카드 드래그 판정과 거래 클릭이 충돌하지 않게 했습니다.
   - 카드 전체 클릭과 키보드 Enter/Space 진입도 유지했습니다.

3. 거래창 미표시 문제
   - 기존 중첩 버튼 구조를 `article + 독립 거래 버튼` 구조로 바꿨습니다.
   - 거래 버튼에서 드래그 포인터 캡처를 시작하지 않게 분리했습니다.
   - `app.js`, `styles.css`, `mobile-first.css`에 배포 버전 쿼리를 추가해 구버전 브라우저 캐시 때문에 수정 기능이 활성화되지 않는 문제를 방지했습니다.
   - 화면 검증 중 전역 글자 크기 컨트롤이 모달 닫기 버튼보다 높은 레이어에서 겹치는 문제를 발견해, 모달 레이어를 안전하게 상향했습니다.

4. 카드 가독성
   - 카드 높이와 폭을 확대했습니다.
   - 섹터명, 회사명, 가격, 등락률, 수익성·안정성·변동성, 보유 정보, 차트 라벨의 글자 크기와 행간을 확대했습니다.
   - 모바일 최저 폭에서도 정보 열이 무너지지 않도록 별도 보정했습니다.

5. 찌라시 활성화
   - 기존 매 턴 찌라시 생성·전송에 더해 새 찌라시 도착 알림을 상단에 표시합니다.
   - 쪽지 미확인 숫자가 증가하며, 정보센터의 찌라시 항목을 누르면 `찌라시` 대화가 바로 열립니다.
   - 실제 브라우저에서 1·2·3턴마다 쪽지 숫자가 1→2→3으로 증가하는 것을 확인했습니다.

6. SURVIVAL INTELLIGENCE 개선
   - 시장과 공시 항목에 클릭 가능한 섹터명을 추가했습니다.
   - 클릭하면 해당 섹터의 거래 모달이 즉시 열립니다.
   - 현재 상승 섹터명은 굵은 붉은색, 하락 섹터명은 굵은 파란색, 보합은 중립색으로 표시합니다.
   - 공식 SEC Form 8-K 주요 사건 분류를 참고해 중요 계약, 자산 취득, 잠정 실적, 재무 의무, 손상 검토, 경영진 변경, 구조조정, 주주환원, 자본 변경, 사업 승인, 운영 리스크, 기타 중요사항 등 12종의 합성 게임 공시를 추가했습니다.
   - 참고: [SEC Form 8-K 공시 항목 최종 규칙](https://www.sec.gov/files/rules/final/33-8400.pdf)

7. 내 정보의 내 목표
   - 현재 실제 순위를 기준으로 `다음 순위 추월`, 필요한 추가 순자산, 진행 막대를 표시합니다.
   - 1위일 때는 `1위 수성`과 2위와의 자산 격차를 표시합니다.
   - 탈락 시에는 다음 게임을 위한 복기 안내로 변경됩니다.

## 미반영 또는 대체 항목

- 미반영 항목은 없습니다.
- 공시는 실제 기업의 최신 공시를 게임에 직접 노출하지 않고, 공식 공시 사건 분류를 바탕으로 만든 합성 문구로 대체했습니다. 실제 기업 오인과 투자 정보 오용을 방지하기 위한 처리입니다.
- 최종 모달 레이어 수정 후 인앱 브라우저가 로컬 주소 접근을 차단해 마지막 재로딩만 수행하지 못했습니다. 해당 레이어 규칙은 자동 UI 계약 테스트와 빌드에서 검증했습니다.
